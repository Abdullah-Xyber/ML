## Dataset Summary

**Description**  
The House Prices Prediction dataset contains detailed records of residential property sales.  
Each row in `train.csv` represents a single house sale and includes structural, physical, and location-related attributes.

The dataset combines:
- Numerical variables  
- Categorical variables  
- Ordinal variables  

This structure makes it suitable for exploratory data analysis and regression modeling.
