# Lab2 Dataset Summary & Problem description
<br>**Description**:The House Prices Prediction Dataset contains detailed information about residential properties and their sale prices. Each record represents a single house sale and includes a wide range of structural, physical, and location attributes that describe the property. The dataset combines numerical, categorical, and ordinal features, providing a comprehensive view of the factors that influence housing prices.

**Problem Definition**:This model focuses on a supervised regression problem that aims to predict the final sale price of a house based on its features. The model learns from historical housing data to estimate prices for new properties.

**Problem Details**: Supervised Learning 

**Regression Target Variable**: Saleprice The model is expected to predict the final sale price of a house (SalePrice) based on its physical characteristics, location, quality, and condition.
